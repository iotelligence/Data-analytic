# Car Counting
## Objective 
```
    To check the changing status of slots over a selected time period. Because we would like to know
    The changing status : Available to Occupied parking.
```
## Prerequisite
```
  1. data in .csv file
  2. python3
```
## Data
```
   - There will be main 4 columns such as Slot which is the parking location, Status is the state of parking (available or occupied), date, and time.
   - We have 16120 data in file to use for counting.
   - We observed the data between morning 9:00am to night 21:00pm.
```
